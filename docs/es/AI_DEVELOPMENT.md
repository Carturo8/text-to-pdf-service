# Documentación de Desarrollo con IA

Este proyecto fue desarrollado como un experimento de ingeniería de software asistido por IA, demostrando las capacidades y colaboración entre supervisión humana y asistentes de codificación con IA.

## Entorno de Desarrollo

**Plataforma**: [Antigravity](https://antigravity.dev) - Entorno de Codificación con IA Agéntica

## Modelos de IA Utilizados (Cronológico)

### 1. Google Gemini 3 Pro (High)
- **Fase**: Configuración inicial del proyecto y diseño de arquitectura
- **Contribuciones**:
  - Scaffolding del proyecto
  - Implementación de arquitectura hexagonal
  - Adaptadores iniciales y capa de dominio

### 2. Claude Sonnet 4.5 (Thinking)
- **Fase**: Desarrollo central y pruebas
- **Contribuciones**:
  - Lógica de conversión PDF
  - Implementación de suite de pruebas
  - Estructura de documentación
  - Configuración Docker

### 3. Claude Opus 4.5 (Thinking)
- **Fase**: Preparación del lanzamiento y funciones avanzadas
- **Contribuciones**:
  - Endpoint de carga multi-archivo
  - Sistema interno de archivado (solo metadatos)
  - Documentación bilingüe
  - Convenciones de commits y documentos de gobernanza

### 4. Google Gemini 3 Flash
- **Fase**: Pulido final y refinamiento de la V1
- **Contribuciones**:
  - Cabeceras de observabilidad y middleware
  - Auditoría de privacidad y limpieza de documentación
  - Branding profesional de Swagger UI
  - Estabilización final de la hoja de ruta

## Filosofía de Desarrollo

Este proyecto sigue un enfoque **Human-in-the-Loop**:
1. Humano define requerimientos y restricciones
2. IA propone e implementa soluciones
3. Humano revisa y proporciona feedback
4. IA itera basándose en feedback
5. Humano aprueba cambios finales

## Transparencia

Este proyecto es transparente sobre la asistencia de IA:
- README indica claramente "experimento asistido por IA"
- Todo el código es revisado y aprobado por el líder humano
- Las contribuciones de IA no están ocultas ni oscurecidas

## Aseguramiento de Calidad

A pesar de la asistencia de IA, el proyecto mantiene:
- ✅ Estándares profesionales de código (PEP 8)
- ✅ Cobertura de pruebas integral (62%+)
- ✅ Cumplimiento arquitectónico estricto (Hexagonal)
- ✅ Documentación bilingüe (EN/ES)
- ✅ Convenciones estándar de la industria

## Declaración de Ética

Este proyecto demuestra que el desarrollo asistido por IA puede producir software de alta calidad y mantenible cuando:
- Se mantiene supervisión humana
- Se prioriza la transparencia
- Se aplican estándares de calidad
- No se compromete la seguridad

# Security Policy

## Supported Versions

Currently, only the latest stable release (v1.x) is supported.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **do not open a public issue**. Instead, please send an email to the project owner.

We aim to respond to all reports within 48 hours.
